# Koriya Bliss Bridal Studio Portfolio

Standalone Vite + React project prepared for Vercel deployment.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
